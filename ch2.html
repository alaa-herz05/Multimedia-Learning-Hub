Long-Answer Questions أسئلة مقالية
Q1 — Multimedia Components
List the hardware and software components required for a multimedia system.

Answer:
A multimedia system requires: Capture Devices such as video cameras, video recorders, microphones, keyboards, mice, graphics tablets, 3D input devices, tactile sensors, VR devices and digitizing hardware; Storage Devices such as hard disks, CD-ROMs and DVD-ROM; Communication Networks such as local networks, intranets, Internet and special high-speed multimedia networks; Computer Systems such as multimedia desktop machines, workstations and MPEG/VIDEO/DSP hardware; and Display Devices such as CD-quality speakers, HDTV, monitors and colour printers.
Arabic
المكونات: أجهزة التقاط + أجهزة تخزين + شبكات اتصال + أنظمة حاسوب + أجهزة عرض.
Q2 — Bitmap Images
Explain how still images are represented digitally.

Answer:
Still images, when uncompressed, are represented as a bitmap, which is a grid of pixels. Images can be input by digitally scanning photographs/pictures or directly from a digital camera. They may be stored using different colour depths such as 1 bit per pixel for black and white, 8 bits per pixel for grey scale or colour map, and 24 bits per pixel for true colour.
Arabic
الصورة الثابتة تُخزن كـ Bitmap، أي شبكة من البكسلات، ويمكن أن تكون 1-bit أو 8-bit أو 24-bit حسب عمق الألوان.
Q3 — Image Size
Compare the size of a 512×512 grey scale image and a 512×512 true colour image.

Answer:
A 512×512 grey scale image using 8 bits per pixel takes about 1/4 MB without compression. A 512×512 true colour image using 24 bits per pixel takes about 3/4 MB without compression. The true colour image is larger because each pixel uses more bits.
Arabic
512×512 رمادي = 1/4 ميغا، و512×512 ألوان حقيقية = 3/4 ميغا بدون ضغط؛ لأن الألوان الحقيقية تستخدم 24 بت لكل بكسل.
Q4 — Audio
What are the main characteristics of digital audio mentioned in the slides?

Answer:
Audio signals are continuous analog signals. They are captured using microphones, then digitised and stored. CD quality audio requires 16-bit sampling at 44.1 kHz. Higher rates such as 24-bit and 96 kHz can be used. Audio is usually compressed, for example using MP3.
Arabic
الصوت إشارة تماثلية مستمرة، يُدخل بالميكروفون ثم يحول ويخزن رقمياً. جودة CD = 16-bit عند 44.1 kHz، وغالباً يُضغط مثل MP3.
Q5 — Audio Storage
Calculate the storage required for 5 minutes of stereo CD-quality audio.

Answer:
According to the slides, 1 minute of stereo CD-quality uncompressed audio requires 10 MB. Therefore, 5 minutes require: 5 × 10 MB = 50 MB.
Arabic
دقيقة واحدة ستيريو = 10 ميغا، إذن 5 دقائق = 5 × 10 = 50 ميغا.
