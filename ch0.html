HTTP, URI & HTML

بروتوكولات ولغات الويب
Slides 3–4 · How the Web works: protocols, languages
HTTP & URI البروتوكول والمعرّف
HTTP (HyperText Transfer Protocol)
A protocol originally designed for transmitting hypermedia, but can also support the transmission of any file type.
URI (Uniform Resource Identifier)
An identifier for the resource accessed (e.g. the host name). Always preceded by the token http://
Arabic
HTTP: بروتوكول نقل النص التشعبي — القواعد التي يتبعها المتصفح والسيرفر لتبادل البيانات. يدعم نقل أي نوع ملف.
URI: معرّف المورد الموحد — "عنوان" الصفحة على الإنترنت، يبدأ دائماً بـ http://
HTML — HyperText Markup Language لغة ترميز النص التشعبي

HTML is a language for publishing Hypermedia on the World Wide Web — defined using SGML. It has two key properties:

1. Uses ASCII → Portable
HTML files use ASCII encoding, making them portable to all different (possibly binary-incompatible) computer hardware.
2. Tag-based Structure
• <token params> — start tag
• </token> — end tag
• Some elements have no ending tag
Basic HTML Page Structure
<HTML> <HEAD> <TITLE>A sample web page.</TITLE> <META NAME="Author" CONTENT="Dr. Saleh Alomari"> </HEAD> <BODY> <P> We can put any text we like here, since this is a paragraph element. </P> </BODY> </HTML>
💡 Next Generation: HTML → XHTML (reformulation of HTML using XML)
Arabic
HTML هي لغة ترميز صفحات الويب تستخدم وسوماً (Tags) لتنسيق المحتوى.
مميزاتها: (1) تستخدم ASCII فهي تعمل على جميع الأجهزة. (2) بنية واضحة: وسم فتح + محتوى + وسم إغلاق.
الجيل التالي: XHTML = HTML بقواعد XML الصارمة.

Multimedia Components & Types

مكونات الوسائط المتعددة وأنواعها
Slides 5–8 · Six components, three types of multimedia
The Six Components of Multimedia المكونات الستة للوسائط المتعددة
1. Text
Words, labels, menus, and hyperlinks. The most basic component.
2. Graphics
Vector or bitmap illustrations, icons, charts, and diagrams.
3. Animation
A sequence of images displayed rapidly to create the illusion of motion. E.g. GIF, Flash.
4. Sound
Audio recordings: speech, music, sound effects. Formats: MP3, WAV, MIDI.
5. Images
Still photographs or scanned pictures. Formats: JPEG, PNG, GIF, TIFF.
6. Videos
Moving images combined with audio. High storage requirement. Formats: MP4, AVI, MOV.
Arabic
المكونات الستة للوسائط المتعددة:
النص (كلمات ووسوم) • الرسوميات (تصميم وتوضيح) • الرسوم المتحركة (تسلسل صور لإيهام الحركة) • الصوت (موسيقى وكلام) • الصور (صور فوتوغرافية) • الفيديو (صور متحركة + صوت)
Types of Multimedia Interaction أنواع التفاعل في الوسائط المتعددة

Interactive multimedia applications fall into two main categories, plus interactive:

TypeDefinitionUser ControlExampleبالعربي
Linear Media Users start at the beginning and progress through a set of sequences until the end. None — fixed order Slide shows, plays وسائط خطية: ترتيب ثابت
Non-linear Media Order of events left to user's discretion. No predefined order; users can enter or exit at any point and any time. Full — user chooses Interactive CD-ROM encyclopedia وسائط غير خطية: المستخدم يختار
Interactive Multimedia User controls when and what elements are delivered. Active — user triggers Video games, e-learning وسائط تفاعلية: المستخدم يتحكم
💡 Key Insight
When interactive multimedia provides a structure of linked elements for navigation → it becomes HyperMedia.
Arabic
Linear (خطي): مثل مشاهدة فيلم — لا تتحكم في الترتيب.
Non-linear (غير خطي): مثل الموسوعة الإلكترونية — تختار ما تريد.
Interactive (تفاعلي): المستخدم يتحكم فيما يظهر ومتى. عندما يضيف التفاعلي روابط تنقل = يصبح Hypermedia.

Authoring Tools & Storage Media

أدوات التأليف ووسائط التخزين
Slides 9–10 · Authoring software, CD-ROM, DVD
Authoring Tools أدوات التأليف

Authoring tools are used to merge multimedia elements into a project.

These software tools are designed to:

  • Manage individual multimedia elements
  • Provide user interaction
Examples: Flash, Adobe Director, Authorware, PowerPoint, HyperCard
Arabic
أدوات التأليف هي برامج تُستخدم لدمج عناصر الوسائط المتعددة في مشروع واحد متكامل وتسمح بالتحكم في التفاعل والتنقل.
CD-ROM & DVD Storage وسائط التخزين

CD-ROM:

  • Most cost-effective distribution medium for multimedia
  • Contains up to 80 minutes of full-screen video or sound
  • CD burners: read discs and convert to audio/video/data formats

DVD (Digital Versatile Disc):

  • Multilayered DVD increases capacity to 18 GB
  • DVD authoring software creates interactive menus for films and games
  • DVD burners: read and convert to audio/video/data formats
Arabic
CD-ROM: الأرخص لتوزيع الوسائط، يتسع لـ 80 دقيقة فيديو.
DVD: سعة أكبر حتى 18 GB بالتعدد الطبقي. يُستخدم للأفلام والألعاب مع قوائم تفاعلية.

Multimedia Applications

تطبيقات الوسائط المتعددة
Slides 11–16 · Business, Schools, Home, Public Places
Where Multimedia Is Used أين تُستخدم الوسائط المتعددة؟
🏢 Business
Presentations, training, marketing, advertising, product demos, databases, catalogues, instant messaging, and networked communication.
Example: Animated instructional multimedia simulates real procedures — trainees practice turning valves and flipping switches safely.
🏫 Schools & Education
Educational software enriches the learning process. Interactive lessons, e-books with embedded media.
Examples: "Just Grandma and Me" for reading skills; medical multimedia (Yale) for physicians to learn new technologies at their own pace.
🏠 Home
Most multimedia projects reach homes via television sets or monitors with built-in user inputs.
Example: Genealogy software "Reunion" — families build family trees with text, images, sounds, and video clips.
🗺️ Public Places
Stand-alone terminals or kiosks provide information and help (airports, malls, museums, hospitals).
Example: Kiosks provide directions, event info, ticket purchasing, and wayfinding.
Arabic
تطبيقات الوسائط المتعددة في أربعة مجالات رئيسية:
الأعمال: عروض تقديمية، تدريب محاكاة، إعلانات
التعليم: برامج تعليمية تفاعلية، مختبرات افتراضية
المنزل: عبر التلفاز أو الشاشة (برامج شجرة العائلة، ألعاب)
الأماكن العامة: أكشاك معلومات في المطارات والمراكز التجارية

Virtual Reality & Chapter Summary

الواقع الافتراضي وملخص الشابتر
Slides 17–18 · VR, VRML, key takeaways
Virtual Reality (VR) الواقع الافتراضي

Virtual Reality is an extension of multimedia. It uses the basic multimedia elements of imagery, sound, and animation.

  • Requires terrific computing power to be realistic
  • Cyberspace is made up of thousands of geometric objects in 3D space
  • Standard for VR on Web: VRML (Virtual Reality Modeling Language)
  • VRML file extension: .wrl
Arabic
الواقع الافتراضي امتداد للوسائط المتعددة يخلق بيئات ثلاثية الأبعاد. يستخدم الصور والصوت والرسوم المتحركة. يتطلب قوة معالجة عالية.
معيار نقله عبر الويب: VRML • امتداد الملف: .wrl
Chapter Summary ملخص الشابتر

Key takeaways from Chapter 0:

  • Multimedia is a combination of text, graphic art, sound, animation, and video
  • Multimedia projects can be linear or nonlinear
  • Multimedia projects are often stored on CD-ROM or DVDs, or hosted on the Web
  • Multimedia is widely used in business, schools, public places, and at home
  • Virtual Reality is an extension of multimedia
Arabic
الوسائط المتعددة تجمع النص والرسوم والصوت والرسوم المتحركة والفيديو.
تكون خطية أو غير خطية. تُحفظ على CD/DVD أو الويب.
تُستخدم في الأعمال والتعليم والمنازل والأماكن العامة.

📋 Exam Questions & Answers — Chapter 0 أسئلة الامتحان والإجابات النموذجية

Q1 — HTTP & URI
What is HTTP? What is URI? How are they related?
✔ Answer: HTTP (HyperText Transfer Protocol) is a protocol originally designed for transmitting hypermedia but can support any file type. URI (Uniform Resource Identifier) identifies the resource being accessed (e.g. host name) and is always preceded by http://. HTTP is the protocol; URI is the address.
Arabic Answer
HTTP: بروتوكول نقل البيانات بين المتصفح والسيرفر، يدعم أي نوع ملف. URI: معرّف المورد = عنوان الصفحة على الإنترنت، يبدأ دائماً بـ http://. HTTP هو البروتوكول، URI هو العنوان.
Q2 — HTML Properties
What are the two key properties of HTML? What is the next generation of HTML?
✔ Answer: (1) HTML uses ASCII encoding → portable to all hardware platforms. (2) HTML uses tags to structure content: opening tag <token> and closing tag </token> (some elements have no ending tag). The next generation is XHTML — a reformulation of HTML using XML.
Arabic Answer
(1) تستخدم ASCII: تعمل على جميع الأجهزة. (2) تستخدم وسوماً: وسم فتح <tag> ووسم إغلاق </tag> وبعضها بدون إغلاق. الجيل القادم: XHTML = HTML بقواعد XML.
Q3 — Six Components
Name the six components of multimedia.
✔ Answer: (1) Text — words, labels, menus. (2) Graphics — illustrations, icons, charts. (3) Animation — rapid image sequence creating illusion of motion. (4) Sound — speech, music, sound effects. (5) Images — still photographs. (6) Videos — moving images combined with audio.
Arabic Answer
(1) النص (2) الرسوميات (3) الرسوم المتحركة (4) الصوت (5) الصور الثابتة (6) الفيديو
Q4 — Linear vs Non-linear
What is the difference between linear and non-linear multimedia? Give an example of each.
✔ Answer: Linear: Users start at the beginning and progress through a fixed sequence until the end — no user control over order. Example: slide show, play. Non-linear: Order of events left to user's discretion — no predefined sequence, users can enter/exit at any point and any time. Example: interactive CD-ROM encyclopedia.
Arabic Answer
خطي: ترتيب ثابت من البداية للنهاية، المستخدم لا يتحكم. مثال: شريط فيديو، عرض شرائح. غير خطي: المستخدم يختار الترتيب، يدخل ويخرج متى يشاء. مثال: موسوعة CD-ROM التفاعلية.
Q5 — CD-ROM vs DVD
Compare CD-ROM and DVD as multimedia storage/distribution media.
✔ Answer: CD-ROM: Most cost-effective distribution medium; holds up to 80 minutes of full-screen video or sound; CD burners read and convert to audio/video/data. DVD: Multilayered technology increases capacity up to 18 GB; DVD authoring software creates interactive menus for films and games; DVD burners read and convert formats.
Arabic Answer
CD-ROM: الأرخص للتوزيع، سعة 80 دقيقة فيديو. DVD: سعة أعلى بكثير حتى 18 GB بالتعدد الطبقي، يدعم قوائم تفاعلية للأفلام والألعاب.
Q6 — Applications of Multimedia
Name four application areas of multimedia and give a specific example for each.
✔ Answer: (1) Business: training simulators letting trainees practice turning valves and flipping switches safely. (2) Schools: medical multimedia (Yale) letting physicians learn new technologies at their own pace. (3) Home: Reunion genealogy software letting families build family trees with text, images, sounds, and videos. (4) Public Places: information kiosks in malls/airports providing directions and services.
Arabic Answer
(1) الأعمال: محاكاة تدريبية آمنة. (2) التعليم: برامج طبية لكلية Yale. (3) المنزل: برنامج Reunion لشجرة العائلة. (4) الأماكن العامة: أكشاك إرشاد في المطارات والمراكز التجارية.
Q7 — Virtual Reality
What is Virtual Reality? What standard is used for VR on the Web? What is the file extension?
✔ Answer: Virtual Reality is an extension of multimedia that uses imagery, sound, and animation to create immersive 3D experiences. It requires terrific computing power. Cyberspace is made up of thousands of geometric objects in 3D space. Standard for VR on the Web: VRML (Virtual Reality Modeling Language). File extension: .wrl
Arabic Answer
الواقع الافتراضي امتداد للوسائط المتعددة يخلق بيئة ثلاثية الأبعاد باستخدام الصور والصوت والرسوم المتحركة. يتطلب قوة حوسبة عالية. معيار نقله عبر الويب: VRML. امتداد الملف: .wrl
