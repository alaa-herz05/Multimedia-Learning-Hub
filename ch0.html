Multimedia — Definition & Core Concepts

تعريف الوسائط المتعددة والمفاهيم الأساسية
Slides 1–8 · What is multimedia, hypertext, hypermedia, WWW
What Is Multimedia? ما هي الوسائط المتعددة؟

Multimedia means that computer information can be represented through audio, graphics, images, video, and animation in addition to traditional media (text).

The word comes from: Multi (= many, more than one) + Media (= the means by which something is communicated to the senses).

Sight
Images, Video, Text, Color
Hearing
Speech, Music, Sounds
Touch
Shape, Texture, Temperature
Arabic Explanation
الوسائط المتعددة تعني استخدام الحاسوب لعرض المعلومات بأكثر من وسيلة في آنٍ واحد: النص والصوت والصورة والفيديو والرسوم المتحركة. الكلمة مكوّنة من: Multi = متعدد، Media = وسيلة اتصال. الهدف هو استثمار حواس الإنسان (البصر، السمع، اللمس) لتوصيل المعلومات بأسلوب أكثر تأثيراً وفاعلية من النص وحده.
Hypertext النص التشعبي

A system meant to be read non-linearly, by following links that point to other parts of the document, or to other documents entirely.

  • Node: a unit containing text, graphics, or sounds
  • Link: a connection between nodes
  • Anchor: the reference point where a link starts
Arabic
النص التشعبي: نظام قراءة غير خطي عبر روابط. تضغط على رابط فينقلك لمكان آخر في الوثيقة أو لوثيقة مختلفة. مكوّن من: Node (عقدة تحوي المحتوى)، Link (الرابط)، Anchor (نقطة الانطلاق).
HyperMedia الوسائط التشعبية

Hypermedia is not constrained to be text-based. It can include other media such as graphics, images, and especially continuous media like sound and video.

The World Wide Web (WWW) is the best and most famous example of a hypermedia application.

👉 Hypermedia = Hypertext + all other media types (images, audio, video) connected by links.
Arabic
الوسائط التشعبية هي امتداد للنص التشعبي لتشمل جميع أنواع الوسائط (ليس النص فقط). مثال: الويب — تنقر على صورة فتفتح مقطع فيديو، أو تنقر على كلمة فتنتقل لصفحة أخرى كاملة.

Arabic
الفرق الجوهري: Multimedia = دمج وسائط متعددة في مكان واحد (قد يكون خطياً). Hypermedia = وسائط متعددة مترابطة بروابط يتنقل بينها المستخدم بحرية (غير خطي دائماً). الويب = أفضل مثال على Hypermedia.

W3C Goals for the WWW أهداف W3C للويب

The World Wide Web Consortium (W3C) has listed the following goals:

  1. Universal access of web resources — by everyone, everywhere.
  2. Effectiveness of navigating available information.
  3. Responsible use of posted material.

HTTP, URI, HTML & XML

بروتوكولات ولغات الويب
Slides 18–22 · How the Web works: protocols, languages, and standards
HTTP & URI البروتوكول والمعرّف
HTTP (HyperText Transfer Protocol)
A protocol originally designed for transmitting hypermedia, but can also support transmission of any file type. It defines how messages are formatted and transmitted between web servers and browsers.
URI (Uniform Resource Identifier)
An identifier for the resource being accessed (e.g., the host name). It is always preceded by the token http:// or https://. Think of it as the address of a web resource.
Arabic
HTTP: بروتوكول نقل النص التشعبي — القواعد التي يتبعها المتصفح والسيرفر لتبادل البيانات. URI: معرّف المورد الموحد — "عنوان" الصفحة على الإنترنت، يبدأ دائماً بـ http:// أو https://. مثال: http://www.google.com هو URI يحدد موقع Google.
HTML — HyperText Markup Language لغة ترميز النص التشعبي

HTML is a language for publishing Hypermedia on the World Wide Web. It is defined using SGML and has two key characteristics:

1. Uses ASCII → Portable
HTML files use ASCII encoding, making them portable to all different (possibly binary-incompatible) computer hardware. Any computer can read an HTML file.
2. Tag-based Structure
HTML uses tags to describe document elements:
• <token params> — opening/start tag
• </token> — closing/end tag
• Some elements have no ending tag
Basic HTML Page Structure (from slide)
<HTML> <HEAD> <TITLE>A sample web page.</TITLE> <META NAME="Author" CONTENT="Dr. Saleh Alomari"> </HEAD> <BODY> <P> We can put any text we like here, since this is a paragraph element. </P> </BODY> </HTML>
💡 Next Generation: HTML → XHTML (reformulation of HTML using XML) → HTML5 (current standard)
Arabic
HTML هي لغة ترميز صفحات الويب. تستخدم وسوماً (Tags) لتنسيق المحتوى. مثلاً: <B>نص غامق</B>. مميزاتها: (1) تستخدم ASCII فهي تعمل على جميع الأجهزة. (2) بنية واضحة: وسم فتح + محتوى + وسم إغلاق. الجيل التالي: XHTML (HTML مع قواعد XML الصارمة) ثم HTML5.
XML — eXtensible Markup Language لغة الترميز الموسعة

XML is a markup language for the WWW in which there is modularity of data, structure, and view. Users or applications can define their own tags (structure).

Key difference from HTML: HTML defines HOW data looks. XML defines WHAT data means.

How XML Works with Databases (from slide example):
  1. First use a global Document Type Definition (DTD) that is already defined
  2. The server-side script abides by DTD rules to generate an XML document from the database according to a user query
  3. Send user the XML Style Sheet (XSL) depending on the type of device used to display the information
Arabic
XML: لغة ترميز موسعة تسمح للمستخدم بتعريف وسوماته الخاصة. الفرق عن HTML: HTML تحدد الشكل، XML تحدد المعنى. مثلاً: <Price>25</Price> — الوسم يوضح أن الرقم سعر. تُستخدم XML مع DTD (لتعريف هيكل البيانات) وXSL (لتحديد طريقة عرضها). W3C اعتمدها عام 1998 كمعيار رئيسي يتجاوز HTML.

Multimedia Components, Types & Authoring

مكونات الوسائط المتعددة وأنواعها وأدوات التأليف
Slides 23–30 · Six components, three types, authoring tools, CD/DVD
The Six Components of Multimedia المكونات الستة للوسائط المتعددة
1. Text
Words, labels, menus, and hyperlinks. The most basic component. Can be formatted, styled, and hyperlinked.
2. Graphics
Vector or bitmap illustrations, icons, charts, and diagrams. Created by software like Illustrator, CorelDraw.
3. Animation
A sequence of images displayed rapidly to create the illusion of motion. Examples: GIF animation, Flash animation.
4. Sound
Audio recordings: speech, music, sound effects. Formats: MP3, WAV, MIDI. Can dramatically enhance a multimedia project.
5. Images
Still photographs or scanned pictures. Bitmap format. Formats: JPEG, PNG, GIF, TIFF.
6. Videos
Moving images combined with audio. High storage requirement. Formats: MP4, AVI, MOV. Usually compressed.
Arabic
المكونات الستة للوسائط المتعددة: النص (كلمات ووسوم)، الرسوميات (تصميم وتوضيح)، الرسوم المتحركة (تسلسل صور لإيهام الحركة)، الصوت (موسيقى وكلام)، الصور (صور فوتوغرافية)، الفيديو (صور متحركة + صوت). كل مكون له طريقة تمثيل وتخزين خاصة في الحاسوب.
Three Types of Multimedia Interaction أنواع التفاعل في الوسائط المتعددة
TypeDefinitionUser ControlExampleبالعربي
Linear Media Users start at beginning and progress through a fixed sequence until the end. No skipping or rearranging. None — fixed order Slide show, movie, play, TV broadcast وسائط خطية: ترتيب ثابت من البداية للنهاية
Non-linear Media Order of events left to user's discretion. No predefined order; users can enter or exit at any point and time. Full — user chooses path Interactive CD-ROM encyclopedia, website وسائط غير خطية: المستخدم يختار المسار
Interactive Multimedia User controls when and what elements are delivered. When linked elements are provided for navigation = becomes hypermedia. Active — user triggers events Video games, e-learning modules, interactive kiosks وسائط تفاعلية: المستخدم يتحكم في التسليم
💡 Key Insight
When interactive multimedia provides a structure of linked elements through which users can navigate, it becomes hypermedia. Interactive multimedia → add navigation links → Hypermedia.
Arabic
Linear (خطي): مثل مشاهدة فيلم — لا تتحكم في الترتيب. Non-linear (غير خطي): مثل الموسوعة الإلكترونية — تختار ما تريد قراءته. Interactive (تفاعلي): المستخدم يتفاعل مع المحتوى ويؤثر فيه. عندما يوفر التفاعلي روابط للتنقل = Hypermedia.
Authoring Tools أدوات التأليف

Authoring tools are used to merge multimedia elements into a project.

These software tools are designed to:

  • Manage individual multimedia elements
  • Provide user interaction
  • Integrate text, graphics, audio, video, and animation
  • Publish to CD-ROM, DVD, or the Web
Examples: Flash, Adobe Director, Authorware, PowerPoint, HyperCard
Arabic
أدوات التأليف هي برامج تُستخدم لدمج عناصر الوسائط المتعددة في مشروع واحد متكامل. تسمح بالتحكم في التفاعل والتنقل وتنظيم المحتوى.
CD-ROM & DVD Storage وسائط التخزين والتوزيع

CD-ROM:

  • Most cost-effective distribution medium for multimedia
  • Contains up to 80 minutes of full-screen video or sound
  • CD burners: read discs and convert to audio/video/data formats

DVD (Digital Versatile Disc):

  • Multilayered DVD increases capacity to 18 GB
  • DVD authoring software creates interactive menus for films and games
  • DVD burners: read and convert to audio/video/data formats
Arabic
CD-ROM: الوسيط الأرخص لتوزيع الوسائط المتعددة. يتسع لـ 80 دقيقة فيديو كامل. DVD: سعة أكبر بكثير (حتى 18 GB) بفضل التعدد الطبقي. يُستخدم للأفلام والألعاب مع قوائم تفاعلية.

Where Multimedia Is Used أين تُستخدم الوسائط المتعددة؟

Presentations, training, marketing, advertising, product demonstrations, databases, catalogues, instant messaging, networked communication.

Stand-alone terminals or kiosks provide information and help in public areas (airports, malls, museums, hospitals).
Example: Kiosks in public places provide directions, event information, ticket purchasing, and wayfinding.

Virtual Reality & Chapter Summary

الواقع الافتراضي وملخص الشابتر
Slides 37–40 · VR, VRML, key takeaways
Virtual Reality (VR) الواقع الافتراضي

Virtual Reality is an extension of multimedia. It uses the basic multimedia elements of imagery, sound, and animation to create immersive 3D experiences.

  • Requires terrific computing power to be realistic
  • Cyberspace is made up of thousands of geometric objects plotted in three-dimensional space
  • Standard for transmitting VR on Web: VRML (Virtual Reality Modeling Language)
  • VRML documents use file extension: .wrl
Arabic
الواقع الافتراضي هو امتداد للوسائط المتعددة يخلق بيئات ثلاثية الأبعاد. يستخدم الصور والصوت والرسوم المتحركة لمحاكاة الواقع. يتطلب قوة معالجة عالية. معيار نقله عبر الإنترنت: VRML بامتداد ملف .wrl.
Chapter Summary (from Slide 39–40) ملخص الشابتر

Key takeaways from Chapter 0 (as stated in the slides):

  • Multimedia is a combination of text, graphic art, sound, animation, and video
  • Multimedia projects can be linear or nonlinear
  • Multimedia projects are often stored on CD-ROM or DVDs, or hosted on the Web
  • Multimedia is widely used in business, schools, public places, and at home
  • Virtual reality is an extension of multimedia
Arabic
ملخص الشابتر: الوسائط المتعددة تجمع النص والرسوم والصوت والرسوم المتحركة والفيديو. تكون خطية أو غير خطية. تُحفظ على CD/DVD أو الويب. تُستخدم في الأعمال والتعليم والمنازل والأماكن العامة. الواقع الافتراضي امتداد لها.

📋 Exam Questions & Model Answers — Chapter 0 أسئلة الامتحان والإجابات النموذجية

Q1 — Definition
Define multimedia. What does "Multi" and "Media" mean?
✔ Answer: Multimedia means that computer information can be represented through audio, graphics, images, video, and animation in addition to traditional media (text). Multi = many/more than one. Media = the means of communicating information to the senses.
Arabic Answer
الوسائط المتعددة تعني استخدام الحاسوب لعرض المعلومات بوسائل متعددة: صوت، رسوم، صور، فيديو، رسوم متحركة إضافة للنص التقليدي. Multi = متعدد. Media = وسيلة اتصال.
Q2 — Hypertext vs HyperMedia
What is the difference between Hypertext and HyperMedia? Give an example of each.
✔ Answer: Hypertext is text-based, read non-linearly by following links between nodes. HyperMedia is not constrained to text — it includes graphics, images, audio, and video linked together. Example: a text document with links = Hypertext. The World Wide Web = HyperMedia.
Arabic Answer
Hypertext: نص فقط مع روابط للتنقل. HyperMedia: جميع أنواع الوسائط (نص + صور + صوت + فيديو) مرتبطة بروابط. مثال على Hypertext: وثيقة نصية بها روابط. مثال على HyperMedia: الويب.
Q3 — WWW Timeline
List the major milestones in the development of the WWW in chronological order.
✔ Answer: (1) 1960s — GML for IBM. (2) 1986 — ISO released SGML. (3) 1990 — HTML and HTTP invented (birth of Web). (4) 1993 — NCSA released Mosaic (first popular browser). (5) 1994 — Netscape Communications formed. (6) 1998 — W3C accepted XML 1.0.
Arabic Answer
1960s (GML) ← 1986 (SGML) ← 1990 (HTML+HTTP — ميلاد الويب) ← 1993 (Mosaic — أول متصفح) ← 1994 (Netscape) ← 1998 (XML). هذا التسلسل مهم جداً للامتحان.
Q4 — HTML Properties
What are the two key properties of HTML? How does it structure content?
✔ Answer: (1) HTML uses ASCII encoding → portable to all hardware platforms. (2) HTML uses tags to structure content: opening tag <token> and closing tag </token>. The next generation is XHTML (HTML reformulated with XML rules).
Arabic Answer
(1) تستخدم ASCII: فهي تعمل على جميع الأجهزة. (2) تستخدم وسوماً (Tags): وسم فتح <tag> ومحتوى ووسم إغلاق </tag>. الجيل القادم: XHTML.
Q5 — Linear vs Non-linear
Differentiate between linear, non-linear, and interactive multimedia with examples.
✔ Answer: Linear: fixed start-to-end sequence, no user control (slideshow, movie). Non-linear: user controls order, can enter/exit at any point (interactive encyclopedia). Interactive: user triggers when/what elements are delivered (games, e-learning). When interactive multimedia provides navigation links → it becomes HyperMedia.
Arabic Answer
خطي: ترتيب ثابت لا تتحكم به (فيلم). غير خطي: المستخدم يختار المسار (موسوعة). تفاعلي: المستخدم يتحكم فيما يظهر ومتى (ألعاب). عندما يضيف التفاعلي روابط تنقل = يصبح Hypermedia.
Q6 — XML
What is XML? How does it differ from HTML? Describe the 3-step process for using XML with databases.
✔ Answer: XML (eXtensible Markup Language) allows users to define their own tags and provides modularity of data, structure, and view. Unlike HTML (defines HOW things look), XML defines WHAT data means. Three steps: (1) Use a global DTD. (2) Server generates XML document following DTD rules. (3) Send user an XSL stylesheet based on device type.

Q7 — Virtual Reality
What is Virtual Reality? What standard is used for VR on the Web? What is the file extension?
✔ Answer: Virtual Reality is an extension of multimedia using imagery, sound, and animation to create immersive 3D experiences. It requires terrific computing power. Standard for VR on the Web: VRML (Virtual Reality Modeling Language). File extension: .wrl
